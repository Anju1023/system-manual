🆘 トラブル対応

全員向け - 困った時の解決策とサポート体制

📋 トラブル対応について

パン屋業務システムで発生するよくあるトラブルの解決方法をまとめています。
基本的な対処法から緊急時の連絡体制まで、困った時にすぐに使える実践的な情報を提供します。

🚨 緊急度別対応

🚨

緊急（即座に対応）

  • 製造ストップにつながるエラー
  • 納品に影響するデータ破損
  • 食品安全に関わる問題
  • システム全体の動作不能
⏰ 即座に責任者に連絡
⚠️

重要（当日中に対応）

  • 一部機能の動作不良
  • 印刷・出力関連の問題
  • データの不整合
  • 作業効率に影響する不具合
📞 営業時間内に相談
💡

通常（翌営業日までに対応）

  • 操作方法の質問
  • レイアウトの調整
  • 新機能の要望
  • 効率化の相談
📝 メールまたは翌日相談

📊 Excel・VBA関連のトラブル

⚡

VBAマクロが動かない

症状：ボタンを押してもマクロが実行されない、エラーが出る

🔍 よくある原因

  • マクロが無効になっている
  • ファイルが破損している
  • メモリ不足
  • 他のプロセスがファイルを使用中

🔧 対処手順

  1. マクロ設定の確認
    ファイル→オプション→トラストセンター→マクロの設定
  2. Excelの再起動
    Excel を完全に終了して再起動
  3. パソコンの再起動
    メモリをクリアして再起動
  4. バックアップファイルで確認
    前日のバックアップで同じエラーが出るか確認
  5. セーフモードで起動
    Excel をセーフモードで起動して確認
🚨 解決しない場合は即座に連絡
🧮

計算結果がおかしい（#N/A、#REF!等）

症状：セルに#N/A、#REF!、#VALUE!などのエラーが表示される

📋 エラーの種類と意味

エラー 意味 主な原因
#N/A 値が見つからない VLOOKUP で参照先が見つからない
#REF! 参照が無効 参照先のセルが削除された
#VALUE! 値の型が間違い 数値のセルに文字が入っている
#DIV/0! ゼロ除算エラー 0で割り算をしている

🔧 対処手順

  1. エラーの種類を確認
    どのエラーが出ているかを確認
  2. 数式の確認
    エラーのセルの数式を確認
  3. 参照先の確認
    VLOOKUP の参照先データを確認
  4. データの整合性確認
    商品名、店舗名の表記統一を確認
  5. 強制再計算
    Ctrl+Shift+Alt+F9 で強制再計算
🔍 #N/Aエラーは商品名の不一致が最多
📂

ファイルが開けない・破損している

症状：Excelファイルが開けない、「ファイルが破損している」と表示される

🔧 復旧手順

  1. 別の方法で開く
    Excel→ファイル→開く→修復して開く
  2. バックアップファイルの確認
    前日・前回保存のバックアップを探す
  3. 自動回復ファイルの確認
    Excel の自動回復ファイルを確認
  4. 別の PC で開く
    他の PC で開けるか確認
  5. 専用ツールでの復旧
    Excel 修復ツールを使用

💾 予防策

  • 定期的なバックアップ（毎日実施）
  • 複数箇所への保存
  • 自動保存機能の有効化
  • ファイルサイズの監視（異常な肥大化に注意）
🆘 復旧不可能な場合は緊急事態
🖨️

印刷がうまくいかない

症状：印刷されない、レイアウトが崩れる、文字が切れる

🔧 対処手順

  1. プリンターの状態確認
    電源、用紙、トナー、エラーランプ
  2. 印刷プレビューで確認
    実際の印刷前にプレビューで確認
  3. ページ設定の確認
    用紙サイズ、余白、拡大縮小設定
  4. プリンタードライバーの更新
    最新のドライバーに更新
  5. PDF経由での印刷
    一度PDFに出力してから印刷
📄 重要な帳票はPDF保存も推奨

💾 システム・データ関連のトラブル

📊

データの不整合・計算が合わない

症状：店舗別の合計と全体の合計が合わない、明らかにおかしい数値

🔍 確認手順

  1. 基本データの確認
    商品データベース、店舗マスターに異常がないか
  2. 最新データの確認
    更新処理が正常に完了しているか
  3. 手計算での検証
    疑問のある数値を電卓で計算
  4. 個別店舗の確認
    異常値を示している店舗データを個別確認
  5. 数式の検証
    SUM、VLOOKUP等の数式が正しいか確認

⚠️ よくある原因

  • 商品名の表記ゆれ（スペース、文字種違い）
  • 終売商品の設定漏れ
  • 新商品のデータ登録不備
  • 店舗コードの重複・間違い
  • 日付の設定ミス
🔍 小さなミスが大きな影響
🐌

Excelの動作が重い・固まる

症状：処理に時間がかかる、応答なしになる、VBAが途中で止まる

🔧 対処手順

  1. 他のアプリケーションを閉じる
    使用していない他のソフトを終了
  2. 自動計算を手動に変更
    数式→計算方法→手動計算に変更
  3. 不要なシートを非表示
    使用していないシートを非表示にする
  4. ファイルサイズの確認
    異常に大きくなっていないか確認
  5. メモリの解放
    Excel を再起動してメモリをクリア

💡 予防策

  • 定期的なファイルの最適化
  • 不要なデータの削除
  • 古いデータのアーカイブ
  • PC のメンテナンス（不要ファイル削除等）
⚡ 定期メンテナンスで予防
💾

バックアップ・復旧

症状：データを間違って削除・変更してしまった、前の状態に戻したい

🔄 復旧手順

  1. 操作の取り消し
    Ctrl+Z で直前の操作を取り消し
  2. 自動回復ファイルの確認
    Excel の自動保存ファイルを確認
  3. バックアップファイルの確認
    定期バックアップからの復旧
  4. バージョン履歴の確認
    OneDrive等のバージョン履歴機能
  5. 手動での再作成
    復旧困難な場合は手動で再作成

💾 バックアップのベストプラクティス

  • 3-2-1ルール：3つのコピー、2つの異なる媒体、1つは別の場所
  • 毎日実施：業務終了時に必ずバックアップ
  • 世代管理：最低1週間分は保持
  • 復旧テスト：定期的に復旧テストを実施
💾 バックアップは業務継続の生命線

📋 業務プロセス関連のトラブル

📞

店舗から急な個数変更依頼

状況：製造開始直前・当日に大幅な個数変更依頼があった

🚨 対応手順

  1. 変更内容の確認
    店舗名、商品名、変更前後の個数、理由
  2. 製造可能性の確認
    材料在庫、製造能力、設備状況
  3. 影響範囲の評価
    他商品への影響、配送への影響
  4. 関係部署への連絡
    製造現場、配送担当、材料発注担当
  5. システムデータの更新
    店舗シートの修正、関連表の更新
  6. 変更記録の作成
    変更内容と対応の記録

⚖️ 判断基準

変更幅 判断 対応
±10%以内 通常対応 在庫・能力内で調整
±20%以内 要検討 製造現場と要相談
±20%超 困難 代替案の提示
⏰ 迅速な判断と関係者への連絡が重要
⚙️

製造設備のトラブル

状況：オーブン故障、ミキサー不調など製造に影響する設備トラブル

🆘 緊急対応手順

  1. 安全確保
    作業者の安全を最優先に確保
  2. 影響範囲の把握
    どの商品・どの工程に影響するか確認
  3. 代替手段の検討
    他の設備での代替可能性を確認
  4. 製造計画の見直し
    システムで製造数・スケジュールを調整
  5. 店舗への連絡
    納品遅延・欠品の可能性を連絡
  6. 復旧作業の開始
    メンテナンス業者への連絡

📋 システム対応

  • 影響商品の製造数を0または削減
  • 代替商品の製造数を増加
  • 発注表の緊急見直し
  • 配送計画の調整
🚨 製造への影響は最小限に
📦

材料不足・納期遅延

状況：重要材料の在庫不足、仕入先からの納期遅延連絡

🔍 対応手順

  1. 在庫状況の正確な把握
    現在庫量と必要量を正確に計算
  2. 製造可能数の算出
    現在庫
