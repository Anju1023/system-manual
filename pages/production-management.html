📊 製造数管理

事務作業者向け - 製造計画と生地出し管理システム

📋 製造数管理システムについて

40店舗への納品個数から最適な製造数を算出し、生地出し計画と天板配置計画を自動作成するシステムです。
商品特性（生地出し日数、成形タイミング）を考慮して、効率的な製造スケジュールを提供します。

製造数シート

2週間分の商品別製造合計数を表示するメインシートです

📋 シートの構成

列 内容 説明
商品名 商品の名称 商品データベースから自動取得
日付列（14日分） 各日の製造必要数 全店舗の納品数を合計
合計 2週間の製造総数 発注量計算に使用

🔍 確認ポイント

  • 異常に多い/少ない製造数がないか
  • 新商品が正しく表示されているか
  • 終売商品が除外されているか
  • 日付の表示が正しいか
📈 このデータが全ての製造計画の基準

生地出しシート（冷凍・スクラッチ）

生地の種類に応じた1週間分の生地出し計画表です

🧊 冷凍生地出し

冷凍生地使用商品の計画

  • 解凍時間を考慮した計画
  • 生地名別に分類

📋 生地出し表の見方

項目 説明 活用方法

商品名 最終的に作る商品 完成品のイメージ
生地個数 必要な生地の個数 実際の作業量
日付 生地出しを行う日 作業スケジュール
⏰ 生地の特性に応じた最適なタイミングで自動計算
📋

並べる表シート

生地を天板に並べる際の詳細な作業指示表です

🎯 並べる表の特徴

  • 5個入り商品は×5で表示
    実際に並べる生地の個数で表示
  • 天板効率を考慮
    最大数と倍数設定で効率的な配置
  • 1週間分の一覧
    先の予定まで見通せる
  • 生地名による分類
    同じ生地の商品をまとめて表示

📊 並べ方の表示例

メロンパン：

  • 並べる数：60個
  • 並べ方：12個乗せが3枚、11個乗せが2枚
  • → 天板を最大限活用した配置指示
🧮 商品設定（最大数・倍数）により自動最適化
👷

夜間用・昼間用シート

製造現場で実際に使用する作業指示表です

🌙 夜間用シート

24時間体制の夜間作業用

  • 夜間に並べる商品のみ表示
  • 数式のみで自動更新
  • リアルタイム反映

☀️ 昼間用シート

日中作業時間帯用

  • 昼間に並べる商品のみ表示
  • ユーザー定義関数使用
  • 並べ方詳細表示

📋 作業表の共通項目

  • 商品名
  • 並べる数（実際の作業量）
  • 生地名（作業分担の目安）
  • 並べ方（天板配置の指示）
  • 納品日（完成目標日）
⚡ 製造数変更時も即座に反映

🔄 製造数更新作業の手順

1

週次定期更新

毎週決まった曜日に実行する標準的な更新作業です

📅 更新タイミング

  • 毎週○曜日の○時（決められた時間）
  • 翌週の製造計画確定後
  • 店舗からの個数変更依頼締切後

📋 更新作業手順

  1. バックアップ作成
    現在のファイルをバックアップ
  2. 店舗別データの確認
    各店舗シートで個数に変更がないか最終確認
  3. 「操作」シートを開く
    VBA操作用のシートに移動
  4. 「更新」ボタンをクリック
    全ての製造関連表を一括更新
  5. 処理完了まで待機
    「更新完了」メッセージが表示されるまで待つ
  6. 結果の確認
    各シートで更新内容を確認
⏰ 標準的な処理時間：3〜5分
2

緊急時の部分更新

急な個数変更や新商品追加時の緊急対応です

🚨 緊急更新が必要な場面

  • 店舗から大幅な個数変更依頼
  • 急な新商品の製造開始
  • 設備トラブルによる製造変更
  • 原材料不足による商品変更

⚡ 緊急更新手順

  1. 変更内容の把握
    どの商品、どの店舗、どの期間の変更か明確化
  2. 影響範囲の確認
    製造・発注・配送への影響を評価
  3. 店舗データの修正
    該当店舗シートで個数を修正
  4. 部分更新実行
    必要最小限の範囲で更新ボタンを実行
  5. 関係部署への連絡
    製造現場、配送、発注担当への情報共有
🚨 緊急時こそ慎重に。影響範囲を必ず確認
3

更新後の確認作業

更新処理後に必ず実施する確認項目です

✅ 必須確認項目

確認項目 確認内容 異常時の対応
製造数シート 合計数が正常か 店舗データを再確認
生地出し表 エラー表示がないか 商品データの整合性確認
並べる表 計算結果が妥当か 商品設定値を確認
夜間・昼間用表 表示内容が適切か 数式の参照先確認

🔍 詳細確認ポイント

  • 数値の妥当性
    異常に大きい/小さい数値がないか
  • 新商品の反映
    新しく追加した商品が正しく表示されているか
  • 終売商品の除外
    終売設定した商品が表示されていないか
  • 日付の整合性
    表示されている日付が正しいか
✅ 確認作業は品質保証の基本
