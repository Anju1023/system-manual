📦 発注管理

事務作業者向け - レシピから自動発注表作成システム

📋 発注管理システムについて

商品の製造数とレシピデータから、必要な材料を自動計算して発注表を作成するシステムです。
材料の無駄を防ぎ、欠品を避けるために正確な発注量を計算します。

🔄 発注システムの仕組み

📊 製造数
各商品の製造予定数
✕
📖 レシピ
商品ごとの材料使用量
⬇️
🧮 VBA自動計算
材料別必要量を集計
⬇️
📄 発注表
🏪 仕入先別発注表
✅ 一度設定すれば、製造数変更時も自動で発注量が更新されます

📚 基礎データ管理

📖

レシピシートの管理

商品ごとの材料使用量を管理するシートです

📋 レシピシートの項目

項目名 説明 入力例
商品名 商品データベースと完全一致させる メロンパン
材料名 材料マスターと完全一致させる 強力粉
使用量 1個作るのに必要な量 0.05
単位 材料の計量単位 kg

✏️ 新しいレシピの追加方法

  1. レシピシートの最下行に移動
  2. 商品名を商品データベースと完全一致で入力
  3. 必要な材料を1行ずつ追加
  4. 使用量は1個あたりの量を入力
  5. 単位は材料マスターと統一
⚠️ 商品名・材料名のスペルミスは計算エラーの原因になります
🧱

材料マスターシートの管理

すべての材料の基本情報を管理するシートです

📋 材料マスターシートの項目

項目名 説明 入力例
材料名 レシピシートと完全一致させる 強力粉
単位 基本的な計量単位 kg
入数 1ケースに入っている数量 25
ケース単位 1ケースの単位 kg
仕入先 発注先の会社名 ○○商事

✏️ 新しい材料の追加方法

  1. 材料マスターシートの最下行に移動
  2. 材料名を入力（レシピで使用する名前と統一）
  3. 単位、入数、ケース単位、仕入先を入力
  4. 発注並び順シートにも追加
💡 入数とケース単位の設定で発注数が自動計算されます
📋

発注並び順シートの管理

実際の発注書と同じ順序で発注表を出力するためのシートです

📄 並び順の重要性

仕入先ごとに、実際の発注書の項目順序と完全に一致させることで、
発注ミスを防ぎ、効率的な発注作業が可能になります。

✏️ 並び順の更新方法

  1. 仕入先から最新の発注書フォーマットを入手
  2. 発注並び順シートを開く
  3. 仕入先別に材料を発注書と同じ順序で並べる
  4. 新しい材料は適切な位置に挿入
🎯 この順序で仕入先別発注表が作成されます

📄 発注表作成手順

1

週次発注表の作成

毎週決まった曜日に実行する標準的な発注表作成です

📋 作成手順

  1. 製造数の確認
    製造数シートで来週の製造予定を確認
  2. VBAボタンの実行
    「操作」シートの「発注表作成」ボタンをクリック
  3. 計算完了の確認
    「計算完了」メッセージが表示されるまで待機
  4. 発注表の確認
    発注シートで計算結果を確認
  5. 仕入先別発注表の確認
    仕入先別発注表シートで最終確認

🔍 確認ポイント

  • 異常に大きな発注数がないか
  • 計算エラー（#N/A、#REF!など）がないか
  • 新商品の材料が正しく計算されているか
⏰ 通常の作成時間：5〜10分
2

緊急時の発注表更新

製造数変更や新商品追加時の緊急対応です

🚨 緊急更新が必要な場面

  • 店舗から大幅な個数変更依頼があった時
  • 新商品の急な追加が決まった時
  • 材料の仕様変更があった時
  • 仕入先の変更があった時

📋 緊急更新手順

  1. 変更内容の確認
    何がどう変わったかを明確にする
  2. 基礎データの更新
    レシピ・材料マスター・製造数を先に更新
  3. 発注表の再作成
    VBAボタンで発注表を再生成
  4. 差分の確認
    前回との差分を必ず確認
  5. 仕入先への連絡
    変更内容を仕入先に速やかに連絡
⚠️ 緊急時こそ慎重に。必ずバックアップを取ってから作業
3

発注表の出力と送付

作成した発注表を仕入先に送付する手順です

📄 出力形式の選択

  • Excel形式：仕入先が編集可能な形式を希望する場合
  • PDF形式：レイアウトを固定したい場合
  • 印刷：FAXや郵送で送付する場合

📧 送付方法

  1. 仕入先別シートの選択
    送付したい仕入先のシートを選択
  2. ファイル名の設定
    「発注表_仕入先名_YYYY-MM-DD」の形式
  3. メール送付
    件名に発注週を明記して送付
  4. 送付記録
    いつ、どこに、何を送ったかを記録
📅 毎週○曜日の○時までに送付完了

🔍 発注表の見方と確認ポイント

📊

発注シートの見方

全材料の総合計が表示

  • 材料名
  • 必要量（kg、個など）
  • ケース単位での発注数
  • 仕入先
📈 全体的な材料使用量を把握
🏪

仕入先別発注表の見方

実際の発注に使用する表

  • 仕入先ごとに分割
  • 発注書と同じ並び順
  • 発注数がケース単位で表示
✅ そのまま発注書として使用可能
⚠️

要確認ポイント

発注前に必ずチェック

  • エラー表示（#N/A等）がないか
  • 異常に多い/少ない発注数
  • 新商品の材料が抜けていないか
  • 単位の間違い
🚨 疑問があれば発注前に要確認

🛠️ よくあるトラブルと対処法

❌

#N/A エラーが表示される

原因：商品名または材料名の不一致

🔧 対処法

  1. エラーが出ている商品名をメモ
  2. 商品データベースで正確な商品名を確認
  3. レシピシートの商品名を修正
  4. 材料名も材料マスターと一致するか確認
  5. 発注表を再作成
🔄

VBAが途中で止まる

原因：データの不整合またはメモリ不足

🔧 対処法

  1. Excelを一度閉じて再起動
  2. 他のアプリケーションを閉じる
  3. バックアップファイルから復旧
  4. データの整合性をチェック
  5. 再度VBAを実行
📊

発注数が明らかにおかしい

原因：レシピの使用量や製造数の設定ミス

🔧 対処法

  1. 該当商品の製造数を確認
  2. レシピの使用量が正しいか確認
  3. 単位（kg、g、個など）が正しいか確認
  4. 材料マスターの入数設定を確認
  5. 手計算で検算してみる

🆘 発注管理に関する重要事項

発注ミスは材料の欠品や過剰在庫につながります。
疑問や不安がある場合は、発注前に必ず確認を取ってください。

特に新商品や大幅な製造数変更時は慎重に対応しましょう。
